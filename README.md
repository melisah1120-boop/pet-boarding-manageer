# 🐾 Pet Boarding Management App

A web-based application for managing pet boarding operations, including check-in, room assignment, and cost calculation.

## 🚀 Features
- Auto-generated pet IDs
- Separate room assignment for dogs and cats
- Room reuse when pets are removed
- Dynamic UI updates using JavaScript
- Boarding cost calculation based on stay duration

## 🛠️ Technologies Used
- JavaScript (Vanilla)
- HTML
- CSS

## 🧠 Project Background
This project was inspired by a Java-based class structure and expanded into a web application to practice DOM manipulation, event handling, and real-world workflow logic.

## 🔄 How It Works
- Pets are added through a form
- Each pet receives:
  - A unique ID
  - A room based on pet type
- When a pet is removed:
  - The room is released and becomes available again
- The UI updates dynamically without page refresh
